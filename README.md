# Planner.

## Начальные требования
- Docker
- Docker-compose
- NodeJs 16+

## Команды для разработки

- Сборка проекта

    `$ make build`

- Запуск проекта

  `$ make start`

- Остановка проекта 

  `$ make stop`

## Вход в приложение

- api http://locahost:5000

- swagger http://locahost:5000/docs

- client http://locahost:3000
